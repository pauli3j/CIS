Control 02: Inventory and Control of Software Assets Safeguards: 7

Why it's critical: Vulnerable or malicious software is a common attack vector. Without an accurate inventory and control process, organizations risk running outdated, unauthorized, or malicious applications. These can be exploited to gain access, move laterally, or execute ransomware and data exfiltration.

Key guidance: Maintain a continuously updated software inventory. Use automated allowlisting, restrict unauthorized installations, and monitor all software activity. Harden systems against software-based attacks using behavioral analysis and approved baseline images.

üß∞ Procedures and Tools

  • Use tools like osquery or AppLocker to monitor and enforce software inventories
  • Implement software allowlisting with WDAC or HashiCorp Nomad
  • Review logs using Elastic or Splunk to detect unauthorized software usage
  • Schedule weekly audits against CIS Benchmarks and image baselines
  • Integrate MDM/UEM for software push & removal (e.g. Jamf, Ivanti)

Control 03: Data Protection Safeguards: 14

Why it's critical: Data is the crown jewel of your organization. It exists across cloud platforms, endpoints, SaaS apps, and partner systems. Without proper classification, encryption, and secure handling, sensitive data can be exfiltrated, ransomed, or accidentally leaked.

Key guidance: Classify data by sensitivity and apply least-privilege access. Enforce encryption at rest and in transit, and secure disposal of physical and digital media. DLP tools and usage audits are your besties here.

üß∞ Procedures and Tools

  • Use VeraCrypt or BitLocker for full disk and file-level encryption
  • Classify data using Microsoft Purview or Varonis
  • Monitor sensitive data movement with DLP tools like Symantec DLP or Forcepoint DLP
  • Securely wipe data with DBAN, BleachBit, or encrypted delete functions
  • Encrypt file transfers with OpenSSH, GnuPG, or secure managed file transfer (MFT) solutions
